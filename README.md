# Onatrix CMS (Umbraco 16)
Onatrix is a CMS-driven website built using Umbraco 16 as part of a .NET development project.  
The focus of the project is to create a clean, responsive, and easily manageable site using Block List components and custom editor configurations.  
This repository contains the full implementation of the front-end structure, Umbraco settings, and form handling logic.

## Technical Overview
- **CMS:** Umbraco 16
- **Languages:** C#, Razor, HTML, CSS
- **Structure:** Block List components, partial views, and global settings via SiteSettings
- **Features:**  
  - Dynamic sections (Hero, Contact, Review, Our Story, etc.)  
  - Custom RTE configuration  
  - Form handling with server- and client-side validation  
  - Mobile-first responsive layout

## Project Structure
- `Views/Partials/BlockList/Components/` – Block components  
- `wwwroot/css/` – Project stylesheets  
- `App_Plugins/` – Custom editor configurations  
- `Controllers/` – Form handling logic
